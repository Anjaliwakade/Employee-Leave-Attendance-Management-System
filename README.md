# Employee Leave & Attendance Management System

## 📖 About Project
This is a full-stack web application built using Spring Boot (Backend) and Angular (Frontend). It manages employee leave requests, attendance tracking, and multi-level approval workflow (Employee → Manager → Admin).

---

## 🚀 Features
- Role-based login (Employee, Manager, Admin)
- Leave request application
- Multi-level approval system
- Attendance tracking (Check-in / Check-out)
- Leave balance management
- Real-time status updates (PENDING_MANAGER, PENDING_ADMIN, APPROVED, REJECTED)

---

## 🛠️ Tech Stack
- Frontend: Angular
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT
- API: RESTful APIs

---

## 👩‍💻 Roles
- Employee: Apply leave, view attendance and leave status
- Manager: Approve/reject employee leave (send to admin)
- Admin: Final approval or rejection

---

## 🔄 Workflow
Employee → PENDING_MANAGER → Manager Approval → PENDING_ADMIN → Admin Approval → APPROVED / REJECTED

---

## 🎯 Purpose
This project is developed to understand real-world HR workflow systems and full-stack development using Java and Angular.
